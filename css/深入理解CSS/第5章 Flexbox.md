# 第5章 Flexbox

弹性盒子布局（Flexible Box Layout）

给元素添加 `display: flex/inline-flex`，该元素变成一个**弹性容器（`flex container`）**，它的直接子元素变成了**弹性子元素（`flex item`）**

`flex` 创建的弹性容器，行为类似 `block` 元素，默认 `width: 100%`，`inline-flex` 则类似 `inline-block`，宽度由内容决定
